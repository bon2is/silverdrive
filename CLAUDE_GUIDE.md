# Project Master Plan: SilverDrive (고령자 운전 적성검사 연습 PWA)

## 1. 프로젝트 개요

- **목표:** 75세 이상 고령 운전자의 면허 갱신용 '인지능력 자가진단' 대비 연습 서비스.
- **핵심 가치:** 실제 검사와 유사한 로직 + 노년층 최적화 UX + 접근성(PWA).
- **수익 모델:** 구글 애드센스(Google AdSense) 광고 수익 기반 무료 서비스.

## 2. 기술 스택 (Tech Stack)

- **Framework:** Next.js 14+ (App Router).
- **Styling:** Tailwind CSS (고대비 모드 및 유연한 레이아웃).
- **PWA:** `next-pwa` (오프라인 접근 및 홈 화면 아이콘 추가).
- **State Management:** Zustand 또는 React Context (사용자 테스트 데이터 관리).
- **Monetization:** Google AdSense Script 통합.

---

## 3. 기능 및 UX 설계 지침

### 👴 노년층 최적화 UX/UI (Crucial)

- **Font Size:** 기본 18px 이상, 제목 24px 이상.
- **Contrast:** 고대비(검정 배경에 노란색 버튼 등) 테마 지원.
- **Interaction:** 더블클릭 배제, 드래그 앤 드롭 지양(단순 클릭/터치 중심).
- **Voice:** 모든 지시사항은 텍스트와 동시에 **브라우저 TTS(Web Speech API)**로 출력.

### 🎮 주요 테스트 로직 (MVP)

1. **자극 반응 테스트:** 화면 중앙에 특정 도형 출현 시 즉시 클릭 (반응 속도 측정).
2. **표지판 식별:** 4~6개의 표지판 중 제시된 이름(예: '진입금지')과 일치하는 아이콘 찾기.
3. **위험 지각:** 도로 주행 사진/영상에서 위험 요소(보행자, 갑자기 튀어나오는 차) 클릭.

---

## 4. 수익 최적화 전략 (AdSense Integration)

### **전략적 대기 화면 (The "Yield" Screen)**

- 테스트 종료 후 `/result` 페이지로 가기 전 `/result-loading` 단계를 강제 삽입.
- **연출:** "어르신의 데이터를 정밀 분석 중입니다..." 애니메이션 (3~5초).
- **배치:** 화면 중앙에 시인성이 높은 **네이티브 광고** 노출.
- **심리:** 결과에 대한 궁금증이 최고조인 시점으로, 광고 노출 시간 및 주목도 극대화.

### **결과 페이지 광고**

- **점수 기반 타겟팅:** 결과 리포트 하단에 관련 정보(보험, 건강식품) 광고 배치.
- **공유 유도:** "자녀에게 결과 보내기" 버튼 클릭 시 전면 광고(Interstitial) 1회 노출 검토.

---

## 5. Claude Code 단계별 개발 로드맵

### **Phase 1: 인프라 및 PWA 세팅**

- Next.js 프로젝트 초기화 및 Tailwind 설정.
- `manifest.json`, `sw.js` 설정으로 홈 화면 추가(A2HS) 기능 구현.
- 어르신용 공통 컴포넌트(큰 버튼, 카드, 텍스트 가이드) 제작.

### **Phase 2: 테스트 엔진 개발**

- `Web Speech API`를 활용한 음성 안내 유틸리티 제작.
- 자극 반응, 표지판 식별 등 개별 게임 로직 및 타이머 구현.
- 전체 테스트 진행 상태 관리 로직 구축.

### **Phase 3: 광고 수익화 페이지 구현**

- `/result-loading` 페이지 디자인 및 타이머 로직(3초 후 결과 자동 이동).
- 애드센스 컴포넌트(`AdBanner`) 제작 및 배치.
- 광고 로딩 실패 시 노출할 '안전 운전 수칙' 카드 뉴스 구현.

### **Phase 4: 결과 리포트 및 분석**

- 사용자 반응 속도를 기반으로 한 "안전/주의/노력" 3단계 결과 알고리즘.
- 카카오톡/문자 결과 공유 기능.
